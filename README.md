# chinua-data-portfolio
Data Analyst | Excel &amp; Google Sheets Specialist
Overview

Chinua Data is a personal portfolio website showcasing data analytics projects, Excel modeling, SQL queries, and other data-driven solutions. The platform demonstrates professional skills in data cleaning, visualization, and analysis.

Features
Project Showcase: Interactive display of completed data projects.
Responsive Design: Optimized for desktop, tablet, and mobile devices.
Detailed Insights: Each project contains downloadable files and descriptions.
Clean UI: Minimalist design focusing on content clarity and usability.
Technologies Used
Frontend: HTML, CSS, JavaScript
Data Visualization: Charts and tables (Excel exports, embedded visualizations)
Deployment: Vercel
Installation / Usage

Clone the repository:

git clone https://github.com/your-username/chinua-data.git
Open index.html in your preferred browser.
To deploy or modify, edit HTML/CSS/JS files and push changes to GitHub.
